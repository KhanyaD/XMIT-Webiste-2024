Disaster Recovery

PREVENTIVE MEASURES

  • Malware prevention via detection software: It is imperative for any modern businesses and organizations to have a way to prevent and identify malware attacks. The malware software must always be updated and well maintained.
  • Security awareness training: It is advised that there are regular security awareness training programs for employees. Various research indicates that these programs are very effective for the reduction of breaches.
  • Policies and procedures: Policies and procedures that are clearly documented are important for ensuring that employees understand internal security controls. It also makes them aware of how dire carelessness is to the organization.
  • User access controls: Limiting who has access to certain data and information is an important security measure for the overall security of the organization. It is not advised that all employees have access to the same information that management has access to.
  • Software patching and updates: To keep the system running seamlessly the software patches must be well maintained. This will also keep the software operating at optimum levels.

    • DETECTIVE MEASURES

  • Change control: Making spontaneous changes to the system will increase the risk of vulnerabilities occurring. This is why when making changes to the system there must be enough planning done so that negative impacts are minimized. Potential threats can be noticed by actively monitoring for unauthorized changes made in the system.
  • Vulnerability Management: Tests can be done regularly for security controls so that the system can remain secure. This way vulnerabilities will be identified early, and they can be delt with accordingly.
  • Incident Altering: Having the mechanisms that allow issues to be detected in a timely manner ensures that an effective method to resolve the issue can be deployed. The impact on an organization is determined by how quickly it is resolved.

    • CORRECTIVE MEASURES

  • 1. The problem must be determined.
  • 2. Determine the magnitude of the problem.
  • 3. Action must be taken to contain the problem.
  • 4. What caused the problem?
  • 5. How is the cause of the problem going to be delt with.
  • 6. The new corrective pan must be put in place.
  • 7. Ensure that there is a follow up process.
